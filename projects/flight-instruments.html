Flight Instruments Demo

Interactive Flight Instruments

Fully functional flight instruments built entirely with Snap SVG and JavaScript. By default, the instruments all go into demo mode with random changes to their properties.

Demo Mode Active: The instruments are automatically animating with random values. Buttons are functional - try clicking the HDG button on the directional gyro. Rotate with the mouse (when not in demo mode) to change values.

Features:

  • Airspeed Indicator
  • Attitude Indicator (Artificial Horizon)
  • Altimeter
  • Turn Coordinator
  • Heading Indicator (Directional Gyro)
  • Vertical Speed Indicator
  • Tachometer (RPM)

These instruments are free and open source for anyone to use without remuneration, attribution or compensation.
