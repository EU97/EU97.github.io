ForceWorkbench

Embedded C++ force measurement system for small aircraft motor thrust characterization

View on GitHub

Overview

ForceWorkbench is an open-source hardware and firmware project for measuring thrust forces from small aircraft motors. Designed to support UAV motor selection and characterization, it provides accurate force measurements essential for control system design and performance optimization.

Built on embedded C++ for microcontroller platforms, the workbench interfaces with load cell sensors, providing real-time data acquisition with serial output for logging and analysis.

Key Features

  • Real-time thrust force measurement via load cell integration
  • C++ embedded firmware for microcontrollers (Arduino/Raspberry Pi Pico compatible)
  • Calibration routines for accurate force readings
  • Serial data transmission for PC-based logging and analysis
  • Support for multiple motor sizes used in UAV platforms
  • Tare, zero-offset, and averaging for stable measurements

Technical Context

Developed to support the UAV platform construction under the ONR Global research project (Grant N62909-20-1-2030). Motor characterization data feeds directly into the control system design and performance models for multi-rotor platforms developed at FIME-UANL.
